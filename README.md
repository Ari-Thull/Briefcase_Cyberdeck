# Briefcase Cyberdeck

A portable cyberdeck built into a briefcase — a self-contained computing/comms unit that opens up into a workstation and closes back into something you can carry.

## Build Timeline

An interactive scrolling timeline of the build lives in [briefcase_cyberdeck_timeline.html](briefcase_cyberdeck_timeline.html). Open it in a browser to step through the project from concept to finished deck.

### Phases

**Concept**
- First control panel test

**V1**
- Panel meets the briefcase
- V1 fully assembled
- In use: working, but flawed

**Pivot**
- Rethinking the architecture after V1

**V2**
- Painted wood base
- Wiring it all together
- Monitor mounted in the lid
- Powered on
- CB radio added

**Final**
- Open: the finished interior
- Closed: portable, sealed

## Viewing

No build step required — just open the HTML file locally:

```
open briefcase_cyberdeck_timeline.html
```

Use arrow keys or the on-screen controls to move through the steps.
